## Some Important Java Concepts

### Java Virtual Machine
The Java Virtual Machine (JVM) is an abstract machine that executes Java bytecode and provides a platform-independent environment for running Java programs.
- JVM is the engine that runs the Java bytecode.
- When you compile Java code, it becomes bytecode (.class files).
- JVM reads the bytecode and executes it on your operating system.<br>

#### Main responsibilities
- Executes Java programs
- Converts bytecode into machine code
- Manages memory (Garbage Collection)
- Provides platform independence ("Write Once, Run Anywhere")
- Handles security and thread management

#### Simple flow

Java Code (.java) --> Compiled by javac --> Bytecode (.class) --> Runs on JVM
